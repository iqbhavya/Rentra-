## Rentra

Rentra is a full-stack rental marketplace where users can explore rental listings and manage property bookings through a simple and responsive web interface. The platform allows users to create, edit, and delete listings while providing an organized system for managing rental properties.


## Features

* Create new rental listings
* View available properties
* Edit existing listings
* Delete listings
* Responsive user interface
* Structured backend with RESTful routes

## Installation

1. Clone the repository
   git clone https://github.com/iqbhavya/rentra.git

2. Move into the project folder
   cd rentra

3. Install dependencies
   npm install

4. Run the server
   node app.js
   
5. Open the browser and visit
   http://localhost:3000
